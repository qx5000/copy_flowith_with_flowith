AI Agent 本地部署平台

基于 Flowith 理念的本地化 AI Agent 系统，支持画布式多智能体协作、知识库管理和工作流编排

快速开始

Docker 部署 (推荐)

# 克隆项目
git clone https://github.com/your-repo/ai-agent-platform.git
cd ai-agent-platform
# 配置环境变量
cp .env.example .env
# 启动服务
docker-compose up --build -d

系统要求

  • 内存: 16GB+ (推荐 32GB)
  • GPU: NVIDIA (8GB+ 显存)
  • 存储: 50GB+ SSD
  • 系统: Docker 支持

核心特性

画布式交互

基于 React Flow 的二维无限画布，支持节点拖拽、连接、参数配置，直观展现 AI 工作流程

多智能体协作

基于 CrewAI + LangGraph 的多 Agent 协同，支持角色定义、任务分工和并行执行

知识库管理

集成 ChromaDB 向量数据库，支持文档上传、自动处理、语义检索和 RAG 增强生成

技术架构

后端技术栈

  • Python + FastAPI
  • LangGraph + CrewAI
  • PostgreSQL + ChromaDB
  • Redis + WebSocket
  • Ollama + LiteLLM

前端技术栈

  • React + Tailwind CSS
  • React Flow 画布
  • Zustand 状态管理
  • WebSocket 实时通信
  • 响应式设计
